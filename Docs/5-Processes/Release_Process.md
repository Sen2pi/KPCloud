# Processo de Release

## Passos
1. Planeamento da versão
2. Desenvolvimento e testes
3. Revisão de código
4. Criação de build
5. Testes finais
6. Deploy em ambiente de produção
7. Comunicação aos utilizadores
8. Documentação de mudanças

## Versionamento
- Semântico (ex: v1.2.3)
- Etiquetas no GitHub
- Changelog atualizado
