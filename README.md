# KPCloud - Gestão de Ficheiros na Nuvem

## Descrição
Sistema de gestão de ficheiros na nuvem com funcionalidades avançadas de upload, download, partilha, gestão de utilizadores e configurações dinâmicas.

## Tecnologias
- Backend: Node.js, Express, MongoDB
- Frontend: React, Material-UI
- Autenticação: JWT
- Armazenamento: Sistema de ficheiros local ou cloud

## Como começar
1. Clonar o repositório
2. Configurar variáveis de ambiente
3. Executar backend e frontend
4. Aceder à interface web
